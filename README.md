# 🌦️ 智能天气 Web 应用技术文档生态作品集

本项目展示了从底层 API 规范制定、结构化 XML 文档编写，到前端 JavaScript 极简交互实现的全流程技术文档工程能力。

## 🌐 线上作品集直达链接
* **[交互式 API 规范手册 (OpenAPI/Swagger)]** -> https://yookynlee.github.io/weather-api-docs/
* **[实时天气查询 Web 应用程序 (JavaScript)]** -> https://yookynlee.github.io/weather-api-docs/app.html

## 🛠️ 所用技术栈与技能点
1. **结构化写作 (Structured Writing)**：熟练应用 **Oxygen XML Author**，基于 **DITA 规范** 编写可复用的 Topic 与 Map 体系。
2. **API 文档工程**：精通 **OpenAPI 3.0 (YAML)** 规范，具备设计标准 RESTful API 接口文档及返回示例（JSON）的能力。
3. **前端技术基础**：掌握基础 **JavaScript (Fetch API)**、HTML5 及 CSS3，具备独立阅读研发代码及排查基本 Web 报错（Error Handling）的技能。
4. **版本控制与托管**：熟练使用 **Git & GitHub** 进行文档版本管理与 GitHub Pages 静态网站自动化部署。

## 📂 仓库文件指南
* `openapi.yaml`: 严格遵循 OpenAPI 规范的接口定义源文件。
* `index.html`: 基于 Swagger UI 渲染的交互式前端文档门户。
* `app.html`: 运用 JavaScript 异步请求（Fetch）实现的交互式天气查询应用。
